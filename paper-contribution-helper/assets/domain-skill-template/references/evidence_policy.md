# Evidence Policy

- Do not fabricate experiments, reviewer comments, replies, baselines, numbers, proofs, citations, or decisions.
- Treat bundled casebook entries as `base-corpus-derived` analogies.
- Treat new PDF evidence as `new-paper-derived`.
- If real reviews are absent, label reviewer attacks as `simulated-review`.
- Mark absent sources as `missing / not reported`; mark unread sources as `not inspected`.
- Keep future-boundary hooks separate from current contributions.
