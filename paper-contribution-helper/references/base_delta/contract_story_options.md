# Contract: Story Option Board

For paper packaging, provide six ranked story-route slots by default unless the user asks for a single route. If fewer than six are defensible from the target evidence, keep the unsupported slots visible and explain why they are unsafe.

Each route includes:

- story name;
- core thesis;
- construction reason;
- evidence anchors;
- manuscript rewrite targets;
- ready-to-paste English snippet when safe;
- anonymous analogy when available;
- best-fit reviewer concern;
- Tier 0 edit path;
- risks and boundaries.

Always include a comparison table over novelty defense, evidence fit, A+B/C resistance, baseline control, mechanism control, cost/reproducibility control, rewrite cost, new-experiment pressure, and best use.

Always include a recommended combination: main route, method/Figure route, related-work/baseline route, experiment-organization route, limitation/discussion route, and minimum Tier 0 route set under page pressure.
