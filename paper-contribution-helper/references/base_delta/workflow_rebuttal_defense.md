# Workflow: Rebuttal Defense

Use for real reviewer comments, rebuttal drafts, or simulated reviewer attacks.

1. Separate real concerns from simulated concerns.
2. Group concerns by novelty, A+B/C, baseline, mechanism, coverage, cost, reproducibility, and overclaim.
3. Trace each concern to likely manuscript triggers.
4. Produce Tier 0 / Tier 1 / Tier 2 repairs.
5. Draft English rebuttal snippets with bounded claims.
